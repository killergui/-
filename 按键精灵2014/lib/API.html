命令名称： 查找窗口句柄
命令功能： 查找窗口句柄
命令参数：
参数1 字符串型，窗口类名
参数2 字符串型，窗口标题
返回值： 整数型，窗口句柄
脚本例子：
按键精灵8及
以上语法 
MsgBox lib.API.查找窗口句柄("notepad",0)
命令名称： 激活窗口并置前
命令功能： 激活窗口并置前
命令参数：
参数1 整数型，窗口句柄
返回值： 无
脚本例子：
按键精灵8及
以上语法 
窗口句柄 = lib.API.查找窗口句柄("notepad",0)
Call lib.API.激活窗口并置前(窗口句柄)
命令名称： 设置窗口透明度
命令功能： 设置窗口透明度
命令参数：
参数1 整数型，窗口句柄
参数2 整数型，透明度（0~255）
返回值： 无
脚本例子：
按键精灵8及
以上语法 
//透明度：0~255
窗口句柄 = lib.API.查找窗口句柄("notepad",0)
Call lib.API.设置窗口透明度(窗口句柄,100)

命令功能： 设置窗口鼠标穿透
命令参数：
参数1 整数型，窗口句柄
返回值： 无
脚本例子：
按键精灵8及
以上语法 
窗口句柄 = lib.API.查找窗口句柄("notepad",0)
Call lib.API.设置窗口鼠标穿透(窗口句柄)
命令名称： 取消窗口设置
命令功能： 取消窗口设置
命令参数：
参数1 整数型，窗口句柄
参数2 整数型，模式（0，一般窗体；1，UpdateLayeredWindow画窗体）
返回值： 无
脚本例子：
按键精灵8及
以上语法 
//模式：0，一般窗体；1，UpdateLayeredWindow画窗体
窗口句柄 = lib.API.查找窗口句柄("notepad",0)
Call lib.API.取消窗口设置(窗口句柄,0)

命令功能： 运行一个指定路径程序
命令参数：

返回值： 无
脚本例子：
